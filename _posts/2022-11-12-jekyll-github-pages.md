title: Как поднять Jekyll+Github Pages
layout: post
date: '2022-11-12 16:40:43'
categories: []
---

1. Установить Ruby через RubyInstaller
2. Установить Jekyll
3. Инициализировать сайт
4. Установить Obsidian, переименовать .markdown в .md
5. Подключить нужные плагины, ориентируясь на совместимость с Github Pages(для превью с YouTube, для тёмной темы, для аналитики)
6. Инициализировать публичный репозиторий на Github
7. Включить в настройках Github Pages
8. В конфиге выставить published и future в true
9. Подождать пару минут(за это время обновится страница на этом домене)
